About

I created this project to find out more about some keywords or buzzwords. When I read something new from a website I always struggle to understand some words as they are new to me. To learn about those I try to search them on google but the problem is you to have to read a lot of articles, courses which can be found on google but what if you could know which resources are already marked as good from others. Will make your life easier, right? And that is the goal of this project. You should be able to find some resources related to a keyword which are already suggested by some users.

This project is open source and crowd sourced. If you want to request any feature or find any bug, please create a issue on Github repository.
